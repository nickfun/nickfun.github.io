MCrypt & PHP

Date Sat 08 March 2014 By Nick Fun

So I was trying to do some work on a side project in Laravel, and I couldn't run anything because it needed the mcrypt PHP extension installed. The artisan utility will not run without mcrypt!

I run Ubuntu everywhere, so I thought I could solve the problem with sudo apt-get install php5-mcrypt and while that command completed successfully, artisan still wouldn't run!

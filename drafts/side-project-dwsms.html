Side Project: dwsms

Date Thu 05 February 2015 By Nick Fun Tags java / dropwizard

At my current job (my last day is tomorrow btw) we are using Dropwizard for the main customer-facing website. I really, really like what Dropwizard is and what it does. It is mostly just a nicely packaged version of existing Java libraries. Having the various components put together is a really nice treat.

Last month I put together my own simple Dropwizard application. It's on my github and you can browse the source if you are interested in how a simple Dropwizard looks. It only has one resource and one view, but it shows how everything fits together.

I call it "dwsms" for a Dropwizard application that sends SMS, it uses twilio for the SMS sending. I got the idea from FutureMe, a nice site that will let you type a letter to yourself and it will wait a few months before sending it. My dwsms project is the same idea, but for text messages. When you run dwsms you get a nice webpage that lets you input a phone number, short message, and a time delay. When the time has passed it will use the twilio API to send the message.
